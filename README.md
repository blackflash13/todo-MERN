## Clone the project to a local machine
```bash
$ git clone
```

## Installation

```bash
$ npm install
```
## Go to server dir
```bash
$ cd server
```

## Fill in the .env file
```
PORT=
DB_URL=
JWT_ACCESS_SECRET=
JWT_REFREST_SECRET=
API_URL=
CLIENT_URL=
```
## Running the app

```bash
$ npm run start

# development
$ npm run dev
```
